# Task 01 - Project Bootstrap

## Goal

Create the initial Go project skeleton and CLI entrypoint.

## Checklist

- [ ] Initialize the Go module.
- [ ] Create the `cmd/codex-switcher` entrypoint.
- [ ] Add the CLI root command.
- [ ] Add command registration for the MVP commands.
- [ ] Add a minimal application package to wire dependencies together.
- [ ] Add a basic config/path resolver package.
- [ ] Add a basic domain model package for shared types and errors.
- [ ] Ensure the project builds from the root.

## Done When

- [ ] `go build ./...` succeeds.
- [ ] The binary starts and prints help or usage text.
- [ ] No warnings or unused declarations are introduced.
