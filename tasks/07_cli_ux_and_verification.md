# Task 07 - CLI UX and Verification

## Goal

Polish command output and run the final verification pass.

## Checklist

- [ ] Make command output concise and consistent.
- [ ] Ensure secret values are never printed.
- [ ] Ensure error messages are actionable.
- [ ] Verify restart guidance text on Linux and macOS.
- [ ] Run `gofmt -w` on modified Go files.
- [ ] Run `go test ./...`.
- [ ] Run `go vet ./...`.
- [ ] Run `golangci-lint run ./...` if available.
- [ ] Fix any warnings, unused variables, or unused imports.
- [ ] Perform a manual smoke test of the CLI commands.

## Done When

- [ ] The project is clean enough to hand off for review.
- [ ] The code passes formatting, tests, vet, and lint checks.
- [ ] No warning-producing code remains.
